# OpenNeuro TBI EEG Falsification Analysis

Public, sanitized release package for:

Artifact-sensitive resting EEG candidate signals in public traumatic brain injury datasets: an OpenNeuro falsification analysis

Author: Gregory Pierpoint, B.S.

ORCID: https://orcid.org/0000-0001-8288-8549

Affiliation: Macon & Joan Brock Virginia Health Sciences Eastern Virginia Medical School at Old Dominion University, Norfolk, VA, USA

Repository URL: https://github.com/blakepi/openneuro-tbi-eeg-falsification

## Contents

- `manuscript/`: Markdown manuscript files, references, citation audit, and editorial checks.
- `reports/`: final decision, reproducibility, audit, and manuscript-readiness reports.
- `outputs/`: selected derived final tables, figure source data, and final figures.
- `scripts/`: analysis and verification scripts needed to understand or reproduce the package workflow.

## Exclusions

This public repository intentionally excludes raw EEG data, DataLad annex contents, virtual environments, credentials, local machine paths where possible, and administrative logs. Raw data remain available from OpenNeuro through the dataset accessions cited in the manuscript.

## Scientific Boundary

This is a null-leaning, artifact-sensitive, reproducibility-oriented public-data analysis. It does not claim a validated biomarker, diagnostic biomarker, clinical deployment, independent validation, confirmed replication, proven EEG signature, or robust positive marker.
